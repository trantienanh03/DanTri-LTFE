# Dân Trí News - React Web Application

Web đọc báo Dân Trí được xây dựng bằng React

## Cấu trúc dự án

```
src/
├── components/        # Các component tái sử dụng
│   ├── BannerMaster/  # Banner component
│   ├── Footer/        # Footer component
│   ├── Header/        # Header component
│   └── NavigationBar/ # Navigation component
├── pages/            # Các trang của ứng dụng
│   └── Home/         # Trang chủ
└── assets/           # Tài nguyên tĩnh
```

## Cài đặt

```bash
npm install
```

## Chạy dự án

### Development mode
```bash
npm run dev
```


## Tác giả

Đồ án LTFE - Dân Trí News Reader
